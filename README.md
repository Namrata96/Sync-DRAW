# Sync-DRAW: Automatic GIF Generation using Deep Recurrent Attentive Architectures

Tensorflow implementation for the paper [Sync-DRAW: Automatic GIF Generation using Deep Recurrent Attentive Architectures](https://arxiv.org/abs/1611.10314) by Gaurav Mittal*,Tanya Marwah* and Vineeth N. Balasubramanian accepted as an oral paper at [ACM-Multimedia 2017](http://www.acmmm.org/2017/) (*Equal Contribution).
